# OCR Detection Chatbot (Image & PDF Support)

An OCR-based chatbot application that extracts text from images and multi-page PDF files with word-level confidence scores.

## Features
- Supports images and multi-page PDFs
- Word-level confidence scores
- Multilingual OCR (English, Telugu, Hindi)
- Image preprocessing using OpenCV
- Interactive web UI using Gradio

## Tech Stack
- Python
- OpenCV
- Tesseract OCR
- Gradio
- NumPy, Pandas

## How to Run
```bash
pip install -r requirements.txt
python app.py
